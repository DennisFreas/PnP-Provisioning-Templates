# Absence and Vacation Requests
This templates provides the basic functionalities for handling HR requests for absence or vacation.
The key features of the template are:
- List of Absence/Vacation Requests
- Custom security group of "HR Managers" that can approve or reject requests
- Approval Workflow for Absence/Vacation Requests
- Customized Home Page with preview of the Absence/Vacation requests
